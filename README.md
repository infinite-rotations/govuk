# GOV.UK Holiday Entitlement Tests

Automated Cypress tests for the **Holiday Entitlement Calculator** pages on GOV.UK  
(https://www.gov.uk/calculate-your-holiday-entitlement)

## Description

This test suite automates the verification of the Holiday Entitlement Calculator pages on GOV.UK using Cypress.
It validates key UI elements, page layout, navigation, sidebars, footers and feedback prompts to ensure the pages render correctly and consistently.

The suite can be run in headed mode for interactive debugging or headless mode for CI/CD pipelines.

## Getting Started

### Installing

1. Clone project: https://github.com/infinite-rotations/govuk.git
2. Run npm i
3. Launch via the relevant script:

Headed mode:

```
npm run cypress:open
```

Headless mode:

```
npm run cypress:run
```
